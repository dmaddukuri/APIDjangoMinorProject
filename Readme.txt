# ConsultAdd Django API Mini Project


This is a Mini Project to demonstrate Sign-up, Sign-in, and host the project in the AWS EC2 instance

## Installation

Use the following commands to install the required software and environment to load and run the Django project

```bash
sudo apt-get install python3-pip
sudo pip3 install virtualenv
virtualenv Djangoproject
source Djangoproject/bin/activate
pip3 install django==2.2

```

## Usage

```python
127.0.0.1:8080 is the url to open the website in local machine
This is hosted in the AWS EC2 instance with ip 3.135.230.195 and port 5050, 
so the url to be used in the local machine to launch the project is 
3.135.230.195:5050
```
